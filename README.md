# Server Monitoring Project

## Overview

A Dockerized monitoring solution built using Python, Prometheus, and Grafana.

## Features

- CPU Monitoring
- Memory Monitoring
- Disk Monitoring
- Network Health Check
- Prometheus Metrics Exporter
- Grafana Dashboard

## Technologies Used

- Python
- Docker
- Docker Compose
- Prometheus
- Grafana
- Linux

## Run Project

```bash
docker-compose up -d
