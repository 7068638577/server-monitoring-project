# Server Monitoring Project

## Project Description

This is a Python-based monitoring project.

The script checks:
- CPU usage
- Memory usage
- Disk usage
- Network connectivity

The script also stores logs in a log file.

---

## Technologies Used

- Python
- psutil
- Linux
- Networking
- Git & GitHub

---

## How To Run

Install dependencies:

```bash
python3 -m pip install psutil
```

Run the script:

```bash
python3 monitor.py
```
